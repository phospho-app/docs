[phospho](https://phospho.ai/) is the open source text analytics platform for LLM apps. Understand your users and turn text into insights.

- Cluster text messages to understand user intents and use cases
- Tag, score and classify new messages
- Setup evaluations to get quantified scores
- AB test your LLM app

**Keywords:** _clustering, automatic evaluations, A/B tests, user analytics_

## How does it work?

<Steps>
  <Step title="Import data">
    **Import messages** to phospho (eg: _what the user asked, what the assistant
    answered_)
  </Step>
  <Step title="Run analysis">
    **Cluster messages and run analysis** on the messages. No code required.
  </Step>
  <Step title="Explore results">
    Visualize results on the **phospho dashboard** and export analytics results with
